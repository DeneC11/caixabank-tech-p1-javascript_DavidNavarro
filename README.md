# Pokédex – Práctica Módulo 1

Este proyecto es una Pokédex creada como práctica del Módulo 1 del Bootcamp de CaixaBank Tech.  
El objetivo es consumir una API pública, generar contenido dinámico y crear páginas individuales para cada elemento.

---

## 🚀 Funcionalidades

- Carga de 50 Pokémon desde la PokéAPI.
- Renderizado dinámico en el `index.html`.
- Generación automática de páginas individuales para cada Pokémon usando Node.js.
- Enlaces desde el índice hacia cada página.
- Diseño básico con CSS.

## 🛠️ Cómo generar las páginas individuales

Ejecutar:

node src/main.js

Esto creará automáticamente los archivos HTML dentro de la carpeta `pokemons/`.
